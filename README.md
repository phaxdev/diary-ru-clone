# Diary.ru Clone

Современная копия Дневник.ру

## Структура
- `backend/` - Node.js + Express
- `frontend/` - React + Vite + Tailwind

## Запуск

```bash
git clone ...
cd diary-ru-clone

# Backend
cd backend
npm install
npm run dev

# Frontend (новый терминал)
cd frontend
npm install
npm run dev
```
